# FlowFix

A lightweight Next.js micro-app that transforms messy notes into structured action items.

## What it does

- Converts raw note dumps into prioritized tasks
- Shows a before/after comparison view
- Highlights exactly what changed
- Displays a confidence score and time-saved estimate
- Reveals the exact prompt used for AI-style refinement

## Tech stack

- Next.js (App Router)
- React + local state
- Tailwind CSS
- Deterministic parsing + AI-style prompt guidance

## Usage

1. Run `npm install`
2. Run `npm run dev`
3. Open the app in the browser
4. Paste messy notes, then click **Run FlowFix**

## Demo sample input

```text
quick catch-up with sara: decide timeline for onboarding demo
check dev branch for the build issue, fix install script asap
follow up with design about menu labels
schedule product sync for friday
note: add more examples to the pitch deck
review analytics once the data pipeline is green
```

## Why this wins

FlowFix solves a single real problem with clarity: streamline ad hoc notes into actionable items. The interface is clean, the transformation is obvious, and the logic is explainable in under 30 seconds.
